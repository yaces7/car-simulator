# 3D Araba Modelleri

Bu klasöre GLB formatında araba modellerini ekleyin.

## Gerekli Dosyalar:
- `sedan.glb` - Sedan araba modeli
- `suv.glb` - SUV araba modeli
- `sports.glb` - Spor araba modeli
- `muscle.glb` - Muscle car modeli
- `supercar.glb` - Süper araba modeli

## GLB Model Kaynakları:

### Ücretsiz Siteler:
1. **Sketchfab** - https://sketchfab.com/
   - "car" ara, filtrelerde "Downloadable" ve "glTF" seç
   - GLB formatında indir (en optimize format)

2. **Poly Pizza** - https://poly.pizza/
   - Basit, düşük poly modeller
   - Direkt GLB formatında

3. **Quaternius** - https://quaternius.com/
   - Ultimate Vehicles Pack
   - Tamamen ücretsiz

4. **Free3D** - https://free3d.com/3d-models/car
   - GLB/GLTF formatında modeller var

5. **CGTrader Free** - https://www.cgtrader.com/free-3d-models/car
   - Filtrede "glTF" seç

## FBX'i GLB'ye Çevirme:
Eğer FBX modeliniz varsa:
- https://products.aspose.app/3d/conversion/fbx-to-glb
- https://anyconv.com/fbx-to-glb-converter/
- https://www.vectary.com/ (import edip GLB olarak export)

## Not:
- Eğer GLB dosyası bulunamazsa, oyun otomatik olarak basit geometrik araba modeli oluşturur
- GLB formatı Three.js için en optimize formattır
- Modellerin pivot noktası merkez olmalı
- Dosya boyutları küçük tutulmalı (tercihen 5MB altı)
